# Teacher Lucky Wheel (FlutLab-ready)

This zip contains only the essential Flutter sources for FlutLab:
- pubspec.yaml
- lib/main.dart

Steps on FlutLab:
1) Create a new Flutter project
2) Replace pubspec.yaml and lib/main.dart with these
3) Click Pub get
4) Run / Build (AAB)

Notes:
- No WebView. Works offline.
- Local storage: SharedPreferences key "teacher_wheel_v9" (same as your web version).
- Multi-language: vi/en/jp.
- Menu: Home / About / Privacy / Online Version (opens https://quay.com.vn).
